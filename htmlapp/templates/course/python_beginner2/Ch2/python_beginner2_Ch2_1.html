    第一題:類別

    這裡我們並不會提到太過深入的類別 class 概念。
    通常 class 裡頭會放入大量的函式 def，尤其是在編寫 函式庫 時， class , def 都會包覆許多的運算以及判斷，而它們就是為了分辨這些功能，而以較為白話的英文命名。 以上一堂課的功能例子來說： 

    class fa:
    def factorial(x):
        for i in range(x-1 , 1 ,-1):
            x=x*i
        return x
test = 5
print(fa.factorial(test))

    上面的例子，我們建立了一個叫做 fa 的 class，而它的裡頭包覆了一項叫做 factorial 的功能，在 class 的外圈，建立了一個變數 test = 5 並將它放入 class 的功能裡頭做運算。
    若放入的不是功能，而是變數，它們也能以 fa.變數 的樣子來使用，這便是 class 的用法。

    練習：

    做個簡單的小練習，創立一個叫做 test 的 class ，裡頭放入一個變數 num=[1,2,3] 試著在 class 外頭 print() 輸出。
