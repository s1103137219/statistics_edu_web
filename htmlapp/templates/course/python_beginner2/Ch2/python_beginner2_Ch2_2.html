    第二題:如何使用函式庫

    在安裝 Python 之後，可以在某一路徑找到 site-packages 資料夾，裡頭存放的資料夾裡頭 .py 檔案，就是以 class , def 包成的套件，可以把 site-packages 當作是函式庫，而通常要安裝都會使用 pip install 套件名稱 這樣的指令，如果是使用 Linux 作業系統就有區分 pip 及 pip3 pip3，這裡就不講述太深。
    通常要使用已安裝的套件，我們會以 import 套件名稱.檔案名.class.def as 自建名稱 這樣階層式的方式引用，比如在此系統有的套件： 

    import numpy as np

print(np.random.randint(0, 3,size=(3,4)))

    我們將 numpy 資料夾引用進來，接著將它命名為 np 呼叫在它裡頭有的 random，並使用 裡頭的功能 randint 建立元素都在 0~3 之間的三乘四二維亂數。

    練習

    引用 numpy，使用看看 random.randint 功能。
