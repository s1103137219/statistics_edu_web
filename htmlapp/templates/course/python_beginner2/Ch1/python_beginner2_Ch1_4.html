    第四題:功能：List相加

    經過 List的特殊宣告(一)、 List的特殊宣告(二) 應該以對 List 有了了解， 我們曾在 功能：判斷型別 提前過， def 名稱(x) 裡頭的未知變數 x 是要放入相對應的資料型別的， 而未知變數其實是會依據功能裡頭運算，來判斷型別的，這也包括了 List , Tuple , Dict , Set。
    例如： 

    def test(x):
    x[0]=x[1]+x[2]
    return x
t=[1,2,3]
print(type(test(t)))

    輸出結果會是 <class 'list'>。
