    第一題:功能：判斷型別

    這裡稱做的功能，其實就是指 副程式(或稱子程式) ，大多數編程人員都稱它為 函數(或函式) ，但在這請先把它當作是程式的某項功能來看待就好。
    Python 的功能開頭是由 def 名稱(): 為起頭的，裡頭的 () 可能會有我們所要運算的值， 以下面的例子來看： 

    def factorial(x):
    for i in range(x-1 , 1 ,-1):
        x=x*i
    return x
z=6
print(factorial(z))

    建立一個讀取各個數字的階層功能，功能的名稱為 factorial，而它 () 可以放入一個未知的數字 x ，而裡頭的 for 迴圈 所做的事情就是算出未知的數字 x 的階層，之後將未知的數字 x 作為結果回傳。
    (假設要算出 5! 階層是以 5*4*3*2*1 這樣的方式，故從 x-1 開始做相乘)
    在功能外直接讓 z=6 接著直接輸出 factorial(z) ，這裡請注意一點 () 裡頭 是放入相對應的資料型別的變數，並不是一定要放入叫做 x 的變數。

    練習

    先做簡單的練習，建立一個功能名稱為 ty，判別未知變數的資料型別 (str , bool , int ,float)。
    建立變數 a = 5 , b = 5.5 , c = "st" , d = False 以 print() 輸出被功能包覆的 a,b,c,d 結果。

    說明

  • 在 def 名稱() 的 () 裡頭先設置一個未知變數，名稱則取代為 ty。
  • 使用 if...else...判斷，如果等於 str , bool , int ,float 回傳(return) 字串、布林、整數、float，如：type(x) == str 便 return "字串"。
  • 請以 print() 作結果的輸出。
