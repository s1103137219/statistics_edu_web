    第二題:List的特殊宣告

    剛剛練習了簡單的編寫功能，這次我們來讓 List 裡頭的元素相加，先看以下的 List 宣告： 

    list1=[0 for x in range(3)]

    裡頭的宣告，先建立了一個元素 0 接著再以 for 迴圈 來敘述要建立三個同樣的元素，也就是說 0 按照了 for 迴圈 的範圍，建立了自己 0,1,2 位置的元素。 不過是只是一維度的宣告，若要使用二維度會是這樣：list1=[ [0 for x in range(3)] for x in range(3) ] ， 這樣的輸出結果會是 [[0,0,0],[0,0,0],[0,0,0]]，先建立了三個 0 的元素後，第二迴圈在這三個 0 的外頭又跑了一次 0,1,2 建立了三次 [0,0,0]，這跟一開始了宣告一樣，只是又包覆了一層 []而已。

    練習

    建立一個二維度的 list1=[ [0 for x in range(3)] for x in range(3) ] 接著請分別將 1,2,3,4,5,6,7,8,9 加入至 list1 的位置。

    說明

  • 一維度的 List 能以 list1[0] 尋找第一個數字的位置， 二維度須以 list1[0][0] 才能捕捉到最裡頭的第一個位置。
  • 此題先禁止使用 for 迴圈 作答。
  • 請以 print() 作結果的輸出。
