    第三題:List的特殊宣告(二)

    若在剛剛的題目中，有在中途 print() 執行程式的話，或許已經理解了二維度的 List了。

    練習

    這次我們延續剛剛的那一題，一樣先建立 list1=[ [0 for x in range(3)] for x in range(3) ] 以及一個累加變數 num = 0 這次我們以 for 迴圈 來將放入 list1 中。

    {% endblock%} {% block comlen%} 進行下一題 再練習一次 {% endblock %} {% block title2%}

    說明

  • 一維度的 List 只需要以一個 for 迴圈 就能獨立讀取所有元素了，二維度則需要兩個 for 迴圈 如下方範例： 
    for i in range(3):
    for j in range(3):
  • 累加變數 num 需放入兩個 for 迴圈 中做 num=num+1 由 0+1,1+1,2+1....8+1 可以當作是三乘三而跑了九次的累加。
  • 請以 print( ) 作結果的輸出。
