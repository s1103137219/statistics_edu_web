    第一題:型別辨識

    在程式語言中，變數是一種存取數值的一種「容器」，它們有著不同的型別，常見的有整數 int、浮點數 float 以及字元 str、布林 bool ……等。

    Python 的變數會依據所存取的數值為何，再藉此定義變數的型別，常見型別例子：
    a = 5，此變數 a 將被視為整數 int
    a = 4.5視為浮點數 float
    a = 'String' 或 a = "String" 視為字元 str
    a = True 或 a = False 視為布林 bool
