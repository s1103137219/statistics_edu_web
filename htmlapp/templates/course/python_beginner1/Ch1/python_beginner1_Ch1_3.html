    第三題:List、Tuple

    Python 中並沒有陣列 Array，但是有類似陣列的存在：List、Tuple， 兩者皆能以一維、二維甚至三維來呈現，如果對維度沒有概念可以暫時參照 維基 在後續新出的課程我們會有題目讓使用者去熟悉維度，入門課程暫且不用有維度概念。
    List 與 Tuple 宣告例子：
    a = [ 1,2,3,4,5 ]，這段為 List 的宣告，裡頭存放了 1,2,3,4,5，這些數值。 若將 [ ] 改為 ( ) 就是 tuple 的宣告了。

    練習

    嘗試使用 List、Tuple 的宣告，並以 list1、tuple1 為變數存取 'A'、'B'、'C' 作為裡頭的元素，並以 print( ) 輸出它們的 'C'。
