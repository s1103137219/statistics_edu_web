    第四題:Dict、Set

    關於 Dict、Set，它們都是以 { } 宣告，但是不同之處在於 Dict 有 Key 以及 Value ，用法與 Json 檔案類似 ，它能夠依照存在的 Key 去找到相對應的 Value ，它也能夠將 List 以及 Tuple 變成自己的型別。
    例子：
    c ={ 1:"name" , 2:"id" }，用 print( c[1] ) 輸出的答案會是 name。
    而 Set 就只是單純的集合， 如果使用 set( 'adc' ) 它就會以 {'a', 'd', 'c'} 將字隔開， 並且將重複的字移除，像 set( 'addcc' ) 裡頭 c、d 重複了，但結果也只會出現一次 變為 { 'a', 'd', 'c' } 。但用到 Set 的機會不多，反而是 dict、List 以及 Tuple 較常出現。

    練習

    請先宣告 a = 'Pyython'、b = [ [1,"t"],[2,"e"] ] ，將 a、b 分別轉為 Set、Dict。
