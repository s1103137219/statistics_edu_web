    第二題:型別轉換

    在不少爬蟲或程式的處理上，我們會將型別做轉換在做其他的處理或是判斷。
    轉換例子：float(5)、int('6')、bool(0)，它們分別輸出的結果就會是 5.0、6、False，但在字元 str 轉換成整數 int 上存在一點特別的因素，如果int()放入的是'S'，便會產生錯誤，需要以 ord() 轉換為整數 int，裡頭的整數 int由來是 Ascii 的對照碼。
    (維基所敘述的 Ascii)

    練習

    接下來，先創立變數 zero = 0、five = 5 轉換為字元 str，再轉為 Ascii　的整數 int ，並將 five、zero 放入print() 輸出。
