    第二題:判斷式

    所謂的判斷，就是以有、沒有來做的決定，在 Python 中判斷式是以 if...else... 所呈現的， 好比判斷 1 < 2 我們雖然能直接以 print() 結果為 True ，但卻不能改變此結果的說法 ，在我們想以其他輸出方式作為判斷的結果便會使用 if...else... 的方式。
    如： 

    if 1 < 2 :
    print("1真的小於2")
else:
    print("1其實不小於2")

    裡頭的例子 print() 之所以要空格是因為 Python 是以類似階層似的方式來判斷是否包覆在裡頭，一般以 空白四格 為縮排做類似階層的動作。 再來上述的例子，只是其中的一種用法，扣除 <、> ，還有代表等於的 == 、不相等的 != 以及其他的 and、or、not 。

    練習：

    創建變數 num = 'a' 、num2 = 'A' 以 if...else...判斷他們誰大誰小， 輸出較大的變數名稱
