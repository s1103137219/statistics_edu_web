    ECDF(經驗累積機率分配函數)

    將數據經過整理之後畫成圖表，便可粗略判斷資料的真實累積分配函數。 因此在本章，將練習到如何編寫 ECDF 函數來整理接收到的資料。

    第五題:編寫ECDF函數

    編寫一個函數 ecdf(data) ，傳回兩個值x和y 。 （ x 為x軸將接收到的 data 陣列由小到大排序， y 為 ECDF 的數據 Y 軸，1/n 以 1 等間距增加）

    numpy 的 sort 可將資料做排序， arange 可以跑 自訂範圍類似 for 。

    說明

  • 變數 n，用 len( ) 計算資料來源的數量。
  • 變數 x，用 np.sort( ) 將資料由小到大排序。
  • 變數 y，用 np.arange( 起始值,終止值 ) ，可做到 1/n 以 1 等間距增加， 但記得 arange 的終止值會跑到 n-1 ，所以終止值要設 n+1 。
  • 最後這個 ECDF 函數記得要 return x,y ，才算是真正完成。
