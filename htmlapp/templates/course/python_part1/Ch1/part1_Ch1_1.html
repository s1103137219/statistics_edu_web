    繪製鳶尾花數據

    本節的練習將使用到植物學家 愛德華．安德森 所收集的資料集， 這些資料集中有三種不同種類的鳶尾花，分別為 山鳶尾(Iris setosa) 、 鳶尾屬(Iris Irisic Iris) 、 維吉尼亞鳶尾(Iris Virginica) 等花瓣長度。

    第一題:繪製直方圖

    利用 matplotlib.pyplot 繪圖套件、 seaborn 加強圖表視覺化套件，繪製50個樣本的花瓣長度的直方圖，存放資料的變數為 versicolor_petal_length 。

    將套件引入時，應該使用 import ，例如： import matplotlib.pyplot as plt ，至於後面的 as 是將引入的套件取個變數名稱，方便後續重複使用時無須再打全名。

    說明

  • import matplotlib.pyplot 套件和 seaborn 套件(通常會將他們的變數名稱取 plt 和 sns )。
  • 使用 seaborn 設定繪製條件。
  • 繪製直方圖可以使用 plt.hist( ) ，裡面放資料來源 versicolor_petal_length 。
  • 顯示出直方圖使用到 plt.show( ) 。
