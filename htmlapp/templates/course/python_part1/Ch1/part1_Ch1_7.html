    進行多種類比較的ECDF圖

    ECDF 圖還可以進行兩個比較甚至更多（但如果過多，圖表會變得混亂）。

    第七題:一次繪製三個物種的ECDF圖

    要三個 ECDF 覆蓋在同一張圖上，可以直接使用 plt.plot( ) 三次 ，記住要包括 marker ='.' 和linestyle ='none' 作為 plt.plot( ) 中的參數。

    {% endblock%} {% block comlen%} 進入下一課 再練習一次 {% endblock %} {% block title2%}

    說明

  • 把 setosa_petal_length、versicolor_petal_length、virginica _petal_length 分別帶入 ecdf( ) 做計算， 並用 x_set, y_set、x_vers, y_vers、x_virg, y_virg 去接收它們各自的回傳值。
  • plot 函數用法， _=plt.plot(X軸資料,Y軸資料,marker='.',linestyle='none') ，此為點狀表示、取消線條連接。本題須做三次。
  • plt.margins( ) 設定 外距 為 0.02 ，預防某些資料會被裁掉。
  • X 與 Y 的標記軸分別為 petal length (cm) 和 ECDF 。
  • plt.legend(('第一個plot', '第二個plot', '第三個plot') , loc='lower right') 可以顯示圖例， loc 能指定圖例所在位置。
  • 畫完要用 plt.show( ) 來顯示。
