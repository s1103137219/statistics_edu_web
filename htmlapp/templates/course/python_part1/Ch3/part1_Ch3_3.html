    第三題:有多少人會拖欠借款？

    假設在某日有 100 人向銀行抵押房屋借款，這些借款拖欠的可能有 0~100 筆不等，拖欠的可能機率為 0.05 。為了調查到底有幾人拖欠借款，將模擬伯努利試驗。

    本次練習將會使用到前一題的函式 perform_bernoulli_trials() ，在這裡機率已預設為 0.05 ，有 100 筆借款 ，除了使用此函式外還需再做 1000 次的模擬。

    {% endblock%} {% block comlen%} 進行下一題 再練習一次 {% endblock %} {% block title2%}

    說明

  • 使用 np.random.seed() 設定值為 42 。
  • 建立變數 n_defaults 儲存用 np.empty() 所產生的陣列，陣列中有 1000 筆空值。
  • 用 for 迴圈來進行 100 筆中有多少人拖欠借款的 1000 次模擬，需使用 perform_bernoulli_trials() ， 依照借款數及預設機率設定 n=100、p=0.05 ，每執行一次，將儲存在 n_defaults 。
  • 繪製 n_defaults 直方圖，參數 normed 設為 True 讓資料以機率的方式呈現。
  • 顯示繪製的直方圖( X、Y標記軸 分別設為 number of defaults out of 100 loans 和 probability )。
