    np.random 與伯努利試驗

    伯努利試驗是指在單一事件中結果只有兩種可能的隨機試驗， 最常見的例子就是猜硬幣，拋上空落下的硬幣會是正面還是反面。 簡單來說就是可以用是或否來回答的事件。

    第二題:np.random 與伯努利試驗

    在本次練習中，將定義一個函式來做試驗，規定函式 perform_bernoulli_trials(n, p) 裡頭的 n 代表試驗的次數， p 代表成功的機率，並使用 np.random.random() 執行每一次的試驗。

    說明

  • 定義函式 perform_bernoulli_trials(n, p) 變數 n_success 設為 0 ， 來記錄試驗成功的次數。
  • 使用 for 迴圈重複執行 n 次試驗。
  • 並用 if 判斷是否成功，若成功就增加次數。以 np.random.random() 是否小於 p 作為條件，符合條件 n_success 便會增加。
  • 將 n_success 回傳 (return) 。
