    共變異數

    第八題:計算共變異數

    使用 np.cov() 計算共變異數，如：有兩份資料集分別為 x 與 y 計算它們的共變異數只要以 np.cov(x , y) 就會回傳一個二維陣列，這個二維陣列就是所謂的共變異數矩陣，位置在 [ 0 , 1 ] 與 [ 1 , 0 ] 的值是 x、y 的共變異數， [ 0 , 0 ] 為 x 的自變異數、 [ 1 , 1 ] 為 y 的自變異數。

    注意：花瓣寬度與長度是有相關性的，包括上一題的題目也是。

    說明

  • 使用 np.cov() 計算鳶尾花花瓣寬度與長度的共變異數矩陣，並存放於變數 covariance_matrix 。
  • 輸出共變異數矩陣。
  • 從共變異數矩陣中取出位置 [ 0 , 1 ] 的共變異數，並以變數 petal_cov 儲存， 請注意 [ 1 , 0 ] 與 [ 0 , 1 ] 的值是一樣的，若有不同就不為共變異數。
  • 輸出共變異數。
