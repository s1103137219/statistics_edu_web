    另一個青年青蛙青蛙 C ，你想看看青蛙 B 和青蛙 C 是否有類似的衝擊力。不幸的是，你沒有青蛙 C 的衝擊力可用，但你知道他們的平均值為 0.55 N 。因為你沒有原始數據，你不能做一個排列測試，你不能評估這個假設，青蛙 B 和青蛙 C 來自相同的分佈。因此，您將測試另一個較少限制性的假設：青蛙 B 的平均擊打力等於青蛙 C 。

    設置引導假說檢定，您將以平均值作為檢驗統計量。記住，你的目標是計算平均衝擊力小於或等於青蛙 B 觀察到的機率，如果青蛙B的衝擊力的真實平均值等於青蛙 C 的真實平均值為真。您首先翻譯青蛙 B 的所有數據，使平均值為 0.55 N 。這包括從青蛙 B 的每次測量中添加青蛙 C 的平均力並減去青蛙 B 的平均力。這留下了青蛙 B 分佈的其他屬性，如方差。
