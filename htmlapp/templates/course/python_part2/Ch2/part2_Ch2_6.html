    說明

  • 使用 draw_bs_pairs_linreg（x，y，size = 1） 定義一個函數來對線性回歸參數執行成對的拔靴法抽樣。
    • 使用 np.arange（） 設置從 0 到 len（x） 的索引數組。 這些是你將重新取樣，並使用它們從 x 和 y 數組中選取值。
    • 使用 empty（） 初始化斜率並截取複製數組的大小。
    • 寫一個 for 循環到：
      • 重新索引指數。 使用np.random.choice（） 來做到這一點。
      • 使用重採樣索引 bs_inds 創建新的 x 和 y 數組 bs_x 和 bs_y 。 要做到這一點，用 bs_inds 切 x 和 y 。
      • 在新的 x 和 y 數組上使用 np.polyfit（） 並儲存計算的斜率和截距。
    • 返回斜率和截距的成對的拔靴法抽樣。
