    一個很好的方式想像我們在線性回歸中可能會發生的變化是繪製從每個拔靴法重複抽樣的斜率和截距得到的線。這樣做是為了前 100 次拔靴法重複抽樣的斜率和截距（存儲為 bs_slope_reps 和 bs_intercept_reps ）。

    說明

  • 生成由 0 和 100 組成的 x 值數組，用於回歸線的圖形。 使用 np.array（） 函數。
  • 寫一個 for 循環，在這個循環中你繪製一個回歸直線，斜率和截距由拔靴確認法的樣本給出。 做 100 行。
    • 當在 for 循環的每次迭代中繪製回歸線時，回顧回歸方程 y = a * x + b 。 這裡， a 是 bs_slope_reps [i] ， b 是 bs_intercept_reps [i] 。
    • 在調用 plt.plot（） 時，指定關鍵字參數 linewidth = 0.5 ， alpha = 0.2 和 color ='red' 。
  • 用 x 軸上的文盲和 y 軸上的生育力做一個點散圖。 請記住指定 marker ='。' 和 linestyle ='none' 關鍵字參數。
  • 標記軸，設置 2％ 的邊距，並顯示圖。 這已經做了，所以點擊“執行”，以看到回歸式！
